#ClothesOne

## サイト概要
ユーザーが自分のお気に入りの服を投稿し、他のユーザーと共有し合えるサイト

### サイトテーマ
良い服を共有しあおう

### テーマを選んだ理由
①服が好きなこと
②前職がアパレルであったこと

### ターゲットユーザ
10代〜50代までの男女

### 主な利用シーン
①自分の好きなアイテムを共有したい時
②他のユーザーの方のアイテムを参考にしたい時
